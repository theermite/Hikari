## Erreurs → cause racine (5 pourquoi) → correction

1. **`cargo build` échoue (vswhom-sys / icu_properties)** → cl.exe exit 1 sur du C++ →
   *pourquoi ?* un job rustc parallèle (icu) a avorté → *pourquoi cl.exe ?* cargo tue les
   jobs en vol quand un autre échoue → **cause racine : échec collatéral, pas un vrai défaut**.
   Correction : re-lancer → exit 0. **Leçon : re-lancer un build parallèle avant de diagnostiquer.**
2. **`React.act is not a function`** (test RTL) → `React.act` = `undefined` en 19.2.7 →
   *pourquoi ?* Testing Library 16.3 attend l'`act` de React, absent de ce build →
   **cause racine : React bleeding-edge en avance sur la lib de test**. Correction : RTL + jsdom
   retirés, test smoke via `renderToStaticMarkup`. **Leçon : vérifier la compat de l'outil de
   test avec la version EXACTE du framework.**
3. **Rust restait en 1.94 après `rustup update stable`** → *pourquoi ?* l'hôte par défaut est
   `gnu`, la chaîne active est `msvc` → `update stable` a mis à jour gnu, pas msvc →
   **cause racine : multi-chaîne, mise à jour de la mauvaise**. Correction : `rustup update
   stable-x86_64-pc-windows-msvc`. **Leçon : sur Windows multi-chaîne, mettre à jour la chaîne ACTIVE.**
